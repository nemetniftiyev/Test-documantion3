
   # BookNest 

## User Stories

1. **User Story:**  
   Mən istifadəçi olaraq istəyirəm ki, "Kitab axtar" düyməsinə klik etdikdə kitab janrları və onlar haqqında məlumat çıxsın.  
   - **Acceptance Criteria:**  
     - "Kitab axtar" düyməsi aktivdir və klik edilə bilir  
     - Klik edildikdə sistem mövcud kitab janrlarını siyahı şəklində göstərir  
     - Hər janr haqqında qısa məlumat görünür  
     - Siyahı həm desktop, həm də mobil görünüşündə düzgün işləyir
2. **User Story:**  
   Mən istifadəçi olaraq istəyirəm ki, seçdiyim kitabın detallı səhifəsinə baxa bilim, müəllif və janr haqqında məlumat əldə edim.  
   - **Acceptance Criteria:**  
     - Kitab seçildikdə detallı səhifə açılır  
     - Səhifədə təsvir, müəllif və janr görünür  
     - Mobil və desktop görünüşündə səhifə düzgün işləyir  

3. **User Story:**  
   Mən istifadəçi olaraq istəyirəm ki, email və şifrə ilə qeydiyyatdan keçə bilim, uğurlu qeydiyyatdan sonra təsdiq mesajı alım.  
   - **Acceptance Criteria:**  
     - Qeydiyyat formu email və şifrə qəbul edir  
     - Uğurlu qeydiyyat zamanı təsdiq mesajı göstərilir  
     - Yanlış email formatı və zəif şifrə üçün xəbərdarlıq çıxır  

4. **User Story:**  
   Mən istifadəçi olaraq istəyirəm ki, login formu ilə profilimə daxil ola bilim və şəxsi məlumatlarımı görüm.  
   - **Acceptance Criteria:**  
     - Login formu aktivdir və input qəbul edir  
     - Uğurlu girişdən sonra profil səhifəsi açılır  
     - Profil səhifəsində istifadəçi məlumatları görünür  

5. **User Story:**  
   Mən istifadəçi olaraq istəyirəm ki, "Qruplar" hissəsində öz rəylərimi bildirim və istəyə uyğun şəkil, media paylaşa bilim.  
   - **Acceptance Criteria:**  
     - "Qruplar" bölməsində rəy formu aktivdir və input qəbul edir  
     - İstifadəçi mətn şəklində rəy əlavə edə bilir  
     - İstifadəçi şəkil və media fayllarını yükləyə bilir  
     - Sistem uğurlu paylaşımı təsdiqləyir  
     - Rəy və media dərhal qrupda görünür
